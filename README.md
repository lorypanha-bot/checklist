# 🎬 Video Complete

A dark, cinematic pre-export checklist for video editors. Run through every element — narrative arc, music, sound effects, motion graphics, speed ramps, and overall feel — before you ship.

**Live demo:** `https://<your-username>.github.io/video-complete`

---

## Features

- **Multi-project support** — create and name as many projects as you need
- **Per-project progress** — each project tracks its own checklist state independently
- **Persistent storage** — everything saves to `localStorage`, survives browser close/reopen
- **6 checklist categories** — Narrative Arc, Music, SFX, Motion Graphics, Speed Ramps, Overall Feel
- **Zero dependencies** — single HTML file, works offline

---

## Checklist Categories

| Category | Items |
|---|---|
| 🎬 Narrative Arc | Hook, structure, ending, callback |
| 🎵 Music | Tone, resolution, rhythm, mix balance |
| 🔊 Sound Effects | Hit points, ambient fill, SFX levels, fades |
| ✨ Motion Graphics | Consistency, timing, outro, readability |
| ⚡ Speed Ramps | Intent, curve smoothness, sync, restraint |
| 🎞 Overall Feel | Emotional loop, pacing, visual consistency, full playback |

## Hosting on GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Save — your site will be live at `https://<your-username>.github.io/<repo-name>`

---

## Local Use

No build step needed. Just open `index.html` in any browser.

```bash
git clone https://github.com/<your-username>/video-complete.git
cd video-complete
open index.html
```
